1. What is an exception in Python?
   - An error that occurs during program execution.

2. Why is exception handling important?
   - It prevents program crashes and improves reliability.

3. Which block contains risky code?
   - The try block.

4. Which block handles errors?
   - The except block.

5. What exception occurs for invalid numeric input?
   - ValueError.

6. What exception occurs when dividing by zero?
   - ZeroDivisionError.

7. Can Python handle multiple exceptions?
   - Yes, using multiple except blocks.

8. What happens if an exception is not handled?
   - The program terminates with an error.

9. Where is exception handling commonly used?
   - User input, file handling, and network operations.

10. How does exception handling improve user experience?
    - By providing clear error messages instead of crashes.
