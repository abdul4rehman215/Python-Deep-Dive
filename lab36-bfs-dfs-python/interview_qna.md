1. **What is graph traversal?**  
   Visiting all nodes of a graph in a systematic way.

2. **What does BFS stand for?**  
   Breadth-First Search.

3. **What does DFS stand for?**  
   Depth-First Search.

4. **Which data structure does BFS use?**  
   A queue.

5. **Which data structure does DFS use?**  
   A stack or recursion.

6. **Which traversal finds shortest paths in unweighted graphs?**  
   BFS.

7. **Which traversal goes deep before backtracking?**  
   DFS.

8. **Give a real-world use case of BFS.**  
   Network broadcasting or shortest path search.

9. **Give a real-world use case of DFS.**  
   Cycle detection or maze solving.

10. **Why are BFS and DFS important in cybersecurity?**  
    They are used in network analysis, attack path discovery, and graph-based security models.
