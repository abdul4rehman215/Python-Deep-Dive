1. **What are command-line arguments?**  
   Inputs passed to a program when it is executed.

2. **Which Python module is used for argument parsing?**  
   argparse.

3. **Why use argparse instead of sys.argv?**  
   It provides validation, help messages, and cleaner code.

4. **What is ArgumentParser used for?**  
   To define and manage command-line arguments.

5. **What are positional arguments?**  
   Arguments that depend on position in the command line.

6. **What are optional arguments?**  
   Arguments prefixed with -- or -.

7. **How do you make an argument mandatory?**  
   By using required=True.

8. **What does --help do automatically?**  
   Displays usage and argument descriptions.

9. **Why is argparse important in real-world tools?**  
   It enables professional CLI applications.

10. **Where is argparse commonly used?**  
    Automation scripts, security tools, DevOps utilities, and CLI apps.
