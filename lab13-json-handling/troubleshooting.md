### Issue: ModuleNotFoundError: No module named 'json'
**Cause:** Corrupted Python installation  
**Fix:** Reinstall Python (json is built-in)

---

### Issue: JSONDecodeError
**Cause:** Invalid JSON syntax in file  
**Fix:** Validate JSON structure (quotes, commas, brackets)

---

### Issue: File not found
**Cause:** JSON file not created yet  
**Fix:** Run the script once before reading the file
