1. What is JSON?
   - A lightweight data format used for data storage and exchange.

2. Which Python module handles JSON?
   - The built-in `json` module.

3. What does json.dumps() do?
   - Converts Python objects into JSON strings.

4. What does json.load() do?
   - Reads JSON data from a file into Python objects.

5. Why is JSON popular in APIs?
   - It is lightweight, readable, and language-independent.

6. What data structure maps naturally to JSON?
   - Python dictionaries.

7. Why use indent in JSON output?
   - To improve readability.

8. Where are JSON files commonly used?
   - Configuration files, APIs, and data storage.

9. Can JSON store lists?
   - Yes, JSON supports arrays.

10. Why is JSON preferred over binary formats?
    - Because it is human-readable and portable.
