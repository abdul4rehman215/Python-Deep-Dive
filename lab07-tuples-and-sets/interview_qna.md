1. What is a tuple in Python?
   - An ordered and immutable collection.

2. How do you define a tuple?
   - Using parentheses ().

3. Why are tuples immutable?
   - To ensure data consistency and safety.

4. What happens when you modify a tuple?
   - A TypeError is raised.

5. What is a set in Python?
   - An unordered collection of unique elements.

6. How are sets defined?
   - Using curly braces {}.

7. What happens when a duplicate value is added to a set?
   - It is ignored.

8. Why are sets useful?
   - For enforcing uniqueness and fast membership checks.

9. When should tuples be used instead of lists?
   - When data should not change.

10. Can sets and tuples store different data types?
    - Yes, both can store mixed data types.
