1. How do you take numeric input from users in Python?
   - Using input() combined with int() or float().

2. Why is int(input()) used instead of input() alone?
   - Because input() returns a string.

3. Which operators perform arithmetic in Python?
   - +, -, *, /, and //.

4. What does the / operator return?
   - A floating-point value.

5. What is integer division?
   - Division that returns only the integer part of the result.

6. What was the result of 20 // 6?
   - 3.

7. Why does Python return a float for division?
   - To preserve precision.

8. When should integer division be used?
   - When fractional values are not needed.

9. What happens if division by zero occurs?
   - A ZeroDivisionError is raised.

10. Where are arithmetic expressions commonly used?
    - Finance, analytics, automation, and algorithms.
