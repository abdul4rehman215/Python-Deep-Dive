1. What is Object-Oriented Programming?
   - A programming paradigm that organizes data and behavior into objects.

2. What is a class in Python?
   - A blueprint used to create objects.

3. What is an object?
   - An instance created from a class.

4. What does the __init__ method do?
   - Initializes object attributes when an object is created.

5. What does the self keyword represent?
   - The current instance of the class.

6. What is an instance attribute?
   - An attribute unique to each object.

7. What is a class attribute?
   - An attribute shared by all objects of a class.

8. How are methods defined in a class?
   - Using functions inside the class definition.

9. Why are classes useful?
   - They improve code organization and reusability.

10. Where is OOP commonly used?
    - Web applications, enterprise software, and system design.
