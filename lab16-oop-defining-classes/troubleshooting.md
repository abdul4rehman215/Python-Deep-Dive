### Issue: AttributeError when accessing attributes
**Cause:** Attribute not initialized in __init__  
**Fix:** Ensure attributes are defined using self.attribute_name

---

### Issue: Method not executing
**Cause:** Method not called on object  
**Fix:** Use object.method_name()

---

### Issue: Confusion between class and instance attributes
**Cause:** Incorrect access method  
**Fix:** Use ClassName.attribute for class attributes
