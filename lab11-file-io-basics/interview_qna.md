1. What is File I/O?
   - Reading data from and writing data to files.

2. Which function opens a file in Python?
   - open()

3. What does mode 'w' do?
   - Writes data and overwrites existing content.

4. What happens if a file does not exist in write mode?
   - Python creates the file.

5. Which mode is used for reading files?
   - 'r'

6. Why is the `with` statement used?
   - It ensures files are closed automatically.

7. How do you read a file line by line?
   - By iterating over the file object.

8. Why is `strip()` used while reading lines?
   - To remove newline characters.

9. How can file contents be verified via terminal?
   - Using the `cat` command.

10. Why is File I/O important?
    - For persistent data storage like logs and configs.
