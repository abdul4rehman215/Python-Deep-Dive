1. **What is PEP 8?**  
   The official Python style guide for writing readable and consistent code.

2. **Why is PEP 8 important?**  
   It improves readability, maintainability, and team collaboration.

3. **What is flake8 used for?**  
   Detecting syntax errors and PEP 8 violations.

4. **Does flake8 modify code?**  
   No, it only reports issues.

5. **What is black?**  
   An opinionated code formatter that enforces consistent style.

6. **What makes black different from autopep8?**  
   Black is strict and opinionated, autopep8 is flexible.

7. **What is autopep8 used for?**  
   Automatically fixing PEP 8 violations.

8. **Give an example of a PEP 8 violation.**  
   Incorrect indentation or missing whitespace.

9. **Why are formatting tools important in production code?**  
   They ensure clean, consistent, and maintainable codebases.

10. **Where are PEP 8 tools commonly used?**  
    In professional Python development and CI/CD pipelines.
