1. **What is a regular expression?**  
   A regex is a pattern used to search, match, or manipulate text.

2. **Which Python module is used for regex?**  
   The built-in `re` module.

3. **Why use re.compile()?**  
   It compiles a pattern once and allows reuse, improving performance and readability.

4. **What does `\d+` represent?**  
   One or more digits.

5. **What does `findall()` return?**  
   A list of all matching patterns found in a string.

6. **How does `sub()` work in regex?**  
   It replaces matched patterns with a specified string.

7. **What are regex groups used for?**  
   To extract specific parts of a matched pattern.

8. **Why use named groups in regex?**  
   They improve readability and clarity when accessing matched values.
