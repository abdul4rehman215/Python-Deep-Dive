1. What are conditionals in Python?
   - Statements that control program flow based on conditions.

2. Which keywords are used for conditional logic?
   - if, elif, and else.

3. What does the if statement do?
   - Executes code when a condition is True.

4. When is elif used?
   - To test additional conditions if previous ones are False.

5. What is the purpose of else?
   - To handle all remaining cases.

6. Why must input be converted before comparison?
   - Because input() returns a string.

7. How does Python evaluate elif blocks?
   - Sequentially from top to bottom.

8. Why does only one block execute?
   - Execution stops after the first True condition.
