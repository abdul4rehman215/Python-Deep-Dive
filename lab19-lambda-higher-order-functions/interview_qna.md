1. What is a lambda function?
   - A small anonymous function written in one line.

2. When are lambda functions commonly used?
   - With map(), filter(), and reduce().

3. What does map() do?
   - Applies a function to each element in a sequence.

4. How was map() used in this lab?
   - To double each number in a list.

5. What does filter() do?
   - Selects elements that meet a condition.

6. How was filter() used?
   - To extract even numbers from a list.

7. What does reduce() do?
   - Reduces a sequence to a single value.

8. Which module provides reduce()?
   - functools.

9. Why are lambda functions useful?
   - They make code concise and readable.

10. When should lambda functions be avoided?
    - When logic becomes complex or unreadable.
