1. **What is the requests library used for?**  
   It is used to send HTTP requests and interact with web APIs.

2. **Which function is used to send an HTTP GET request?**  
   The requests.get() function.

3. **What does a status code 200 mean?**  
   The request was successful.

4. **How do you access the status code of a response?**  
   Using response.status_code.

5. **How is JSON data extracted from a response?**  
   By calling response.json().

6. **What type of object does response.json() return?**  
   A Python dictionary.

7. **Why is the GitHub API useful for practice?**  
   It is public and returns structured JSON data.

8. **Why is the get() method used on dictionaries?**  
   To safely access values without raising KeyError.
