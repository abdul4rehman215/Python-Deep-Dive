1. What is a function in Python?
   - A reusable block of code that performs a specific task.

2. Which keyword defines a function?
   - def.

3. What is a parameter?
   - A variable that receives data when a function is called.

4. What does the return statement do?
   - Sends a value back to the caller.

5. How do you call a function?
   - By using the function name followed by parentheses.

6. What is a default parameter?
   - A parameter with a predefined value.

7. Why are default parameters useful?
   - They allow functions to run without explicit arguments.

8. What are keyword arguments?
   - Arguments passed using parameter names.

9. How do functions improve code quality?
   - They improve reuse, readability, and maintenance.

10. Where are functions commonly used?
    - Automation, data processing, and application logic.
