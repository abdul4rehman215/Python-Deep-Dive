1. **What is a CSV file?**  
   A CSV file stores tabular data where values are separated by commas.

2. **Which Python module is used to handle CSV files?**  
   The built-in csv module.

3. **What does csv.reader() return?**  
   Each row as a list of strings.

4. **How do you skip the header row in a CSV file?**  
   By calling next() on the csv reader.

5. **What is csv.DictReader()?**  
   It reads CSV rows as dictionaries using headers as keys.

6. **Why is DictReader preferred in real projects?**  
   It allows accessing values by column names.

7. **Are CSV values read as strings or numbers?**  
   They are read as strings by default.

8. **Where is CSV processing commonly used?**  
   Data analysis, automation scripts, ETL pipelines, and reporting tools.
