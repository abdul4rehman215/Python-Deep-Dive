1. **What is unit testing?**  
   Unit testing verifies that individual units of code behave as expected.

2. **Which Python module is used for unit testing?**  
   The built-in `unittest` module.

3. **What is unittest.TestCase?**  
   It is a base class used to define and organize test cases.

4. **How does unittest identify test methods?**  
   Methods starting with `test_` are automatically treated as tests.

5. **What does assertEqual() do?**  
   It checks whether the expected and actual values are equal.

6. **Why is unittest.main() used?**  
   It runs the test suite when the script is executed directly.

7. **What does a dot (.) in test output indicate?**  
   It represents a successfully passed test.

8. **Why is unit testing important in real projects?**  
   It helps detect bugs early and ensures code reliability.
