1. **What is parameter passing in Python?**  
   Passing values to a function when it is called.

2. **What is *args used for?**  
   To accept a variable number of positional arguments.

3. **What is **kwargs used for?**  
   To accept a variable number of keyword arguments.

4. **What type of object is *args?**  
   A tuple.

5. **What type of object is **kwargs?**  
   A dictionary.

6. **When should *args be used?**  
   When the number of positional arguments is not fixed.

7. **When should **kwargs be used?**  
   When functions need flexible named parameters.

8. **What is argument unpacking?**  
   Expanding lists or dictionaries into function arguments.

9. **How do you unpack a list into a function?**  
   Using the * operator.

10. **Why are *args and **kwargs important in real-world Python code?**  
    They allow writing flexible, reusable, and extensible functions.
