1. What is a module in Python?
   - A module is a Python file containing reusable code.

2. How do you import a module?
   - Using the import keyword.

3. Why are modules important?
   - They improve code organization and reuse.

4. What is a package in Python?
   - A directory that contains related modules.

5. What file makes a directory a package?
   - __init__.py.

6. How do you import a module from a package?
   - Using dotted notation.

7. Why are packages used in large projects?
   - For scalability and maintainability.

8. What does import mymodule do?
   - Makes all public members of the module available.

9. How does modular code help teams?
   - It improves collaboration and reduces conflicts.

10. Where are modules commonly used?
    - Libraries, frameworks, and enterprise applications.
