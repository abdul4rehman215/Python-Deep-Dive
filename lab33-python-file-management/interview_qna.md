1. **What is file management scripting?**  
   Automating file operations like copying, renaming, and listing files using scripts.

2. **Which Python modules are commonly used for file management?**  
   os and shutil.

3. **What does os.listdir() do?**  
   Lists all files and directories in a given folder.

4. **Why use os.path.join()?**  
   To safely construct file paths across operating systems.

5. **What is the purpose of shutil.copy()?**  
   To copy files from one location to another.

6. **How do you check if a directory exists in Python?**  
   Using os.path.exists().

7. **How do you create a directory programmatically?**  
   Using os.makedirs().

8. **Why automate file backups using scripts?**  
   To save time and reduce human error.

9. **How do you rename a file in Python?**  
   By modifying the filename before copying or using os.rename().

10. **Where is file automation commonly used?**  
    System administration, DevOps, security operations, and backups.
