1. What is a list in Python?
   - A list is an ordered and mutable collection of elements.

2. How do you define a list?
   - By using square brackets [].

3. Which method adds an item to the end of a list?
   - append().

4. What does the remove() method do?
   - Removes the first matching element from the list.

5. What happens if remove() is called on a non-existent item?
   - Python raises a ValueError.

6. How does the sort() method work?
   - It sorts elements in ascending order by default.

7. Does sort() create a new list?
   - No, it modifies the original list in place.

8. How do you loop through a list?
   - Using a for loop.

9. Can lists store multiple data types?
   - Yes, lists can store mixed data types.

10. Why are lists important in real-world Python programs?
    - They efficiently manage collections of related data.
