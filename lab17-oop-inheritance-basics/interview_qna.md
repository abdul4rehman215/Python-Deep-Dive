1. What is inheritance in OOP?
   - A mechanism where a class inherits attributes and methods from another class.

2. What is a parent class?
   - A class whose properties are inherited by another class.

3. What is a child class?
   - A class that inherits from another class.

4. How is inheritance defined in Python?
   - By passing the parent class name in parentheses.

5. What does the super() function do?
   - It allows access to methods and attributes of the parent class.

6. Why was super().__init__() used?
   - To initialize attributes from the parent class.

7. What is method overriding?
   - Redefining a parent class method in a child class.

8. Why is inheritance useful?
   - It promotes code reuse and reduces duplication.

9. How was inheritance demonstrated in this lab?
   - By extending Car into ElectricCar.

10. Where is inheritance used in real-world systems?
    - Frameworks, APIs, enterprise applications, and secure architectures.
