1. What is a list comprehension?
   - A concise way to create lists in a single line.

2. What is the syntax of a list comprehension?
   - [expression for item in iterable]

3. What was created using list comprehension in this lab?
   - A list of squared numbers.

4. What is a dictionary comprehension?
   - A compact way to create dictionaries.

5. What is the syntax of dictionary comprehension?
   - {key: value for item in iterable}

6. Why are comprehensions preferred over loops?
   - They are cleaner, shorter, and easier to read.

7. Can conditions be added to comprehensions?
   - Yes, using if statements.

8. Are comprehensions faster than loops?
   - Often slightly faster due to internal optimizations.

9. Where are comprehensions commonly used?
   - Data processing and transformations.

10. When should comprehensions be avoided?
    - When logic becomes complex or unreadable.
